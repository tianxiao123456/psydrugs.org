# 抗抑郁药

本分类包含各种抗抑郁药物的信息和使用指南。

- 舍曲林
- 氟西汀
- 西酞普兰
- 氟伏沙明
- 文拉法辛（SNRI）
- 地文拉法辛（SNRI）
- 托鲁地文拉法辛（SNDRI）
- 度洛西汀（SNRI）
- 伏硫西汀
- [右美沙芬](/drugs/解离剂/右美沙芬_愈美片)
- 曲唑酮
- [安非他酮](/drugs/抗ADHD药物/安非他酮) -尽管他是NDRI
